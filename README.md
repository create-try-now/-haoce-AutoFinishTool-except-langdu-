# 好策（HaoCe）平台自动化脚本

基于 Python 的好策平台自动化工具，支持自动登录、书籍任务进度检测、自动发帖/回帖、AI 内容生成及校验模拟。  
内置联网搜索增强接口，可在生成内容前注入搜索结果。朗读任务因语音合成（VITS）调节复杂，暂不支持自动化，需用户自行完成。

## 功能特性

- 自动登录好策平台，获取已加入的书籍列表
- 三种运行模式：
  - **检测模式**：手动查看任务进度、讨论区帖子、章节内容等
  - **自动模式**：真实执行发帖、回帖、报告、摘抄、概要等操作
  - **校验模式**：不发送请求，仅展示 AI 生成的内容，用于安全预览
- 集成 AI 内容生成（兼容 OpenAI Chat Completions 协议，如 Ollama）
- 支持自定义系统提示词，可微调生成风格
- 可选联网搜索增强（需自行搭建搜索接口）
- 内置中英文模板，AI 不可用时自动降级

## 环境要求

- Python 3.10+
- 依赖库（使用 `pip` 安装）：
  - `requests`
  - 需要额外安装 `ffmpeg` 并将其加入系统 PATH。
